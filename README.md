# ass_2_part2

## Summary
This project implements a reinforcement learning solution using Q-learning. It includes a greedy algorithm, instance generation tools, training scripts, and test suites for validating states and stopping criteria. Performance evaluation is provided through `performance_exc1.py`.

## Project Structure

- **greedy/** - Greedy algorithm implementation
- **instancegenerator/** - Tool for generating problem instances
- **performance_exc1.py** - Performance evaluation script
- **requirements.txt** - Project dependencies
- **test_states.py** - State validation tests
- **test_stop_criterion.py** - Stopping criterion tests
- **training_q.py** - Q-learning training script
